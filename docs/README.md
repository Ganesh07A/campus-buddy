# CampusBuddy
CampusBuddy is a modern college portal designed to centralize
academic communication between administration and students.

## Problem Statement
Colleges often rely on physical notice boards or scattered
communication channels, making it difficult for students to stay
updated on notices, events, and academic resources.

## Solution CampusBuddy provides a digital platform where:
- Admins can upload notices, events, and PYQs
- Students can access updates in real time
- Information is centralized and easy to manage

## Key Features
- Role-based access (Admin / Student)
- File-based notices (PDF / PNG)
- Event management
- Previous Year Question (PYQ) repository
- Secure authentication
- Responsive dashboard with dark/light mode

## Tech Stack
- Frontend: Next.js (App Router)
- Backend: Next.js API Routes
- Database: Supabase (PostgreSQL)
- Storage: Supabase Storage
- Auth: Cookie-based authentication
- UI: Tailwind CSS + ShadCN
